Linux Git installation
=====================
cmd : yum install git -> git installation commands
cmd : yum remove git  -> git unistallation
cmd : git config --list -> List of config settings
cmd : got config --global user.name <name>
cmd : got config --global user.email <email>

windows git : git version 2.31.0.windows.1

Linux git   :git version 1.8.3.1

** All commands in git begins with keyword got
